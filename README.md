# surge-rules

个人 Surge 分流规则集，用于按域名把流量固定到指定代理节点，避免自动测速/故障转移导致出口 IP 频繁跳变。

## 背景

Surge 在多个节点间切换时，导致同一账号的服务（如 Claude、Google）在不同地理位置的节点间跳动。部分服务的风控系统会把这种"不可能旅行"模式判定为异常登录，强制要求重新验证身份。本仓库通过固定域名到固定节点的方式规避此问题。

## 文件说明

| 文件 | 用途 |
|---|---|
| `claude.list` | Claude / Anthropic 相关域名 |
| `google.list` | Google / YouTube 相关域名 |
